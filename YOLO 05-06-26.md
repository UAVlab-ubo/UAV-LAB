YOLO (You Only Look Once) sirve principalmente para detectar objetos en imágenes y videos en tiempo real. A diferencia de otros sistemas que analizan una imagen por partes, YOLO la procesa completa de un solo golpe, lo que lo hace extremadamente rápido y eficiente.
## Funciones principales

* Detección de objetos: Localiza elementos dentro de una imagen (como personas, autos, animales) y dibuja una caja (bounding box) alrededor de ellos. 
* Clasificación: Identifica qué tipo de objeto es el que ha encontrado dentro de la caja.
* Segmentación de instancias: Delinea la forma exacta del objeto píxel por píxel, no solo con un cuadro. 
* Seguimiento (Tracking): Sigue el movimiento de un objeto específico a lo largo de un video y le asigna un identificador único (útil para contar personas o vehículos).
* Estimación de postura (Pose Estimation): Identifica la posición del cuerpo humano detectando puntos clave como articulaciones (hombros, rodillas, codos). 

## Casos de uso comunes

* Vehículos autónomos: Para detectar peatones, señales de tráfico y otros autos en la calle.
* Sistemas de seguridad: Para vigilancia automática, detección de intrusos o monitoreo de áreas restringidas.
* Control de aforo: Para contar de forma automática cuántas personas o vehículos entran y salen de un lugar.
* Industria: Para inspección de calidad en líneas de producción o conteo de inventario en almacenes. 


La versión de YOLO que más se utiliza actualmente en producción y proyectos es YOLOv8, debido a su gran equilibrio entre velocidad y precisión, excelente documentación y facilidad de uso.
Sin embargo, dependiendo de tu caso de uso, aquí tienes las opciones principales:
* **YOLOv8:** La más recomendada por su estabilidad, gran comunidad y soporte para detección, segmentación, clasificación y estimación de poses.
Versiones más recientes: Existen versiones más nuevas como **YOLOv11** y **YOLO26**, las cuales ofrecen mayor precisión y menor cantidad de parámetros, ideales para servidores o dispositivos de borde.
* **YOLOv5:** Sigue siendo ampliamente utilizada para investigar o entrenar modelos personalizados por su gran eficiencia y código estable.

# Otra instalación de YOLO 👻



YOLO (You Only Look Once) es uno de los algoritmos más rápidos y populares en inteligencia artificial para la detección de objetos en tiempo real (personas, vehículos, animales, etc.). 
La forma más sencilla y moderna de usarlo es mediante la biblioteca oficial Ultralytics en Python, que permite hacer predicciones con solo unas pocas líneas de código. 
## 1. Instalación
Para comenzar, abre tu terminal o línea de comandos y asegúrate de tener instalado Python. Luego, instala la biblioteca oficial ejecutando:
```c
pip install ultralytics
```
## 2. Uso básico (Detección en una imagen)
Puedes utilizar un modelo preentrenado para detectar objetos en cualquier imagen. El modelo descargará automáticamente los pesos estándar (como el dataset COCO) la primera vez que lo ejecutes.
Crea un archivo de Python (por ejemplo, detectar.py) y escribe lo siguiente:
```c
from ultralytics import YOLO
# Cargar un modelo preentrenado (por ejemplo, YOLOv8 o YOLOv11)model = YOLO('yolo11n.pt')
# Ejecutar la detección en una imagen
results = model('tu_imagen.jpg', save=True, conf=0.5)
# Mostrar los resultados
results[0].show()
```

* **`'yolo11n.pt'`**: Es el modelo más pequeño y rápido de YOLO (Nano). Puedes cambiarlo por otras variantes como `yolo11s.pt` (Small) o `yolo11m.pt` (Medium) si prefieres mayor precisión a cambio de velocidad.
* **`save=True`**: Guarda una nueva imagen con los cuadros delimitadores (bounding boxes) dibujados.
* **`conf=0.5`**: Establece un umbral de confianza del 50% para evitar detecciones erróneas.

## 3. Detección en video o webcam
YOLO brilla especialmente por su velocidad en video. Para procesar un archivo de video o activar la cámara de tu computadora, puedes usar este código:
```c
import cv2from ultralytics import YOLO
model = YOLO('yolo11n.pt')
# Cargar video (usa 0 para usar la cámara web en vivo)cap = cv2.VideoCapture('tu_video.mp4') 
while cap.isOpened():
    success, frame = cap.read()
    
    if success:
        # Ejecutar el modelo YOLO en el fotograma actual
        results = model(frame)
        
        # Visualizar los resultados en el fotograma
        annotated_frame = results[0].plot()
        
        # Mostrar el video
        cv2.imshow("Detección YOLO", annotated_frame)
        
        # Salir si se presiona la tecla 'q'
        if cv2.waitKey(1) & 0xFF == ord("q"):
            break
    else:
        break

cap.release()
cv2.destroyAllWindows()
```
## 4. ¿Qué más puedes hacer?
Ultralytics no solo permite detectar objetos, sino también otras tareas avanzadas de visión por computadora:

* Segmentación: Encontrar la forma exacta de los objetos (máscaras poligonales).
* Clasificación: Decir qué es una imagen en su totalidad.
* Seguimiento (Tracking): Contar objetos y seguir su trayectoria entre varios fotogramas.
