# Cómo usar dron DJI avata 2 para principiantes
Usar el DJI Avata 2 es muy intuitivo gracias al DJI RC Motion 3. 
Para empezar: despliega el dron, carga las baterías, enciende el control y las gafas, conéctalos a la app DJI Fly para activarlos, y despega desde el suelo o la mano presionando dos veces el botón naranja.
## 1. Preparación y Activación
   - Carga: Asegúrate de que las baterías del dron, las gafas (DJI Goggles) y el control estén al 100%.
   - Activación obligatoria: Antes de tu primer vuelo, enciende los equipos y conecta el puerto USB-C de las gafas a tu celular. Abre la app DJI Fly y sigue las instrucciones para vincular el dron a tu cuenta.
   - Seguridad: Quita los protectores del gimbal y despliega las hélices con cuidado.
## 2. Modos de Vuelo para Principiantes
   - Modo N (Normal): Ideal para principiantes. El dron se mantiene en vuelo estacionario (hovering) si sueltas los mandos y cuenta con sensores anticolisión activos.
   - Modo S (Sport): Mayor velocidad y maniobrabilidad, pero con menos asistencias.
   - Modo M (Manual): Para pilotos expertos (modo acrobático/acro).
   - Acro Fácil (Easy ACRO): Te permite hacer giros de 180°, flips y rolls de forma automática mediante el joystick, sin tener que pilotar en modo manual completo.
## 3. Controles Básicos (con RC Motion 3)
   - Despegue/Aterrizaje: Presiona dos veces el botón naranja para encender los motores. Mantenlo presionado para despegar automáticamente. Para aterrizar, mantén presionado el mismo botón.
   - Elevación e inclinación: El gatillo en la parte frontal del control hace que el dron avance; al soltarlo, se detiene y frena.
   - Giro y desplazamiento: Mueve el joystick principal hacia los lados para girar o trasladar el dron en esa dirección.
## 4. Consejos Esenciales
   - Botón de Freno (Pause): Si sientes que pierdes el control o vas a chocar, presiona el botón amarillo (Freno/Estacionario) en el control. El dron se detendrá inmediatamente y se quedará flotando en el lugar.
   - Modo Tortuga: Si el dron aterriza boca abajo por accidente, este modo utiliza los motores para girarlo y dejarlo listo para despegar de nuevo.
