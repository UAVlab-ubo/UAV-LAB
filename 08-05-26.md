# Ollama

Para poder instalar Ollama desde la Powershell se necesita este comando:
```c
irm https://ollama.com/install.ps1 | iex
```
https://ollama.com/library/llama3.1
```c
ollama run llama3.1
```
Si se quiere conectar a OpenClaw Ollama 
```c
bash
ollama launch openclaw
```
(Si no se tiene OpenClaw se instalará de forma inmediata a través de la NPM (se necesita tener instalado Note.js))
