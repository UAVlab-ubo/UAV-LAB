# Descargar Ubuntu 24.04 en WSL para Powershell
Abrir Powershell como administrador y escribir:
```c 
wsl --install -d Ubuntu-24.04
```

# Comandos de la forma estándar de configurar el acceso remoto y el escaneo de redes en un sistema Debian/Ubuntu.
Instala tanto el cliente SSH (para conectarte a otros equipos) como el servidor SSH (para permitir que otros se conecten al tuyo).
```
sudo apt-get install ssh
```

Instala específicamente los componentes del servidor. Al terminar, tu computadora podrá recibir conexiones remotas entrantes.
```
sudo apt install openssh-server
```

Instala una herramienta potente de seguridad de red utilizada para descubrir dispositivos y encontrar puertos abiertos.
```
sudo apt install nmap
```
