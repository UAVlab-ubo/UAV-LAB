# Descargar Ubuntu 24.04 en WSL para Powershell
Abrir Powershell como administrador y escribir:
```c 
wsl --install -d Ubuntu-24.04
```

---

## 1. Instalación de Herramientas de Conexión Remota (SSH)
```c
sudo apt-get install ssh
```
Instala el paquete completo de SSH. Incluye el cliente (para conectarte a otros equipos) y el servidor (para recibir conexiones).

```c
sudo apt install openssh-server
```
Instala específicamente el componente de servidor. Permite que tu máquina acepte conexiones remotas de forma segura.

## 2. Gestión del Servicio SSH
```c
sudo systemctl start ssh
```
Enciende el servidor SSH en este momento. Activa la escucha de conexiones entrantes de inmediato.

```c
sudo systemctl enable ssh
```
Configura el servidor SSH para que se inicie automáticamente cada vez que la computadora se encienda.

## 3. Red y Diagnóstico (Net-Tools)
```c
sudo apt install net-tools
```
Instala un conjunto clásico de herramientas de red para Linux. Incluye comandos antiguos pero muy usados como ifconfig, netstat y route.

```c
sudo ifconfig
```
Muestra la configuración de red actual de tu equipo. Te permite ver la dirección IP local (en la línea que dice inet), la máscara de subred y las interfaces activas.

## 4. Escaneo de Seguridad (Nmap)
```c
sudo apt install nmap
```
Instala el escáner de redes Nmap. Sirve para descubrir qué dispositivos están conectados a una red y qué puertos tienen abiertos.

```c
sudo nmap [IP de inet]
```
Analiza tu propia computadora utilizando su dirección IP. Te muestra qué puertos y servicios (como el SSH que acabas de activar) están abiertos y expuestos en la red.

**EN CMD** conectar cliente a servidor con conexion ssh
```
ssh -p [puerto abierto en ssh, en este caso 22] usuario@direccion_ip
```

Para conectar dos computadoras usando WSL y PowerShell, debes configurar el PC 2 como Servidor (el que recibe la conexión) y el PC 1 como Cliente (el que controla).
Aquí tienes la guía paso a paso para lograrlo y acceder a los archivos.
------------------------------
## Paso 1: Configurar el PC 2 (Servidor / El que será controlado)
Debes asegurarte de que el servidor SSH esté corriendo en Windows o dentro de WSL. La forma más estable en Windows es usar el SSH nativo del sistema.

  **1. Abre PowerShell como Administrador en el PC 2 y ejecuta:**
  ```c
  Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0
  ```
  **2. Inicia el servicio y configúralo para que arranque automático:**
  ```c
  Start-Service sshd
  Set-Service -Name sshd -StartupType 'Automatic'
  ```
  **3. Obtén la dirección IP local de este PC 2 ejecutando:**
  ```c
  ipconfig
  ```
  (Busca la sección "Dirección IPv4", por ejemplo: 192.168.1.15).

------------------------------
## Paso 2: Conectar desde el PC 1 (Cliente) usando WSL o PowerShell
Ve al PC 1. Puedes abrir PowerShell o tu terminal de WSL (Ubuntu), el comando es exactamente el mismo.

**1. Ejecuta el comando de conexión reemplazando con los datos del PC 2:**
  ```c
  ssh usuario_pc2@IP_del_pc2
  ```
  * `usuario_pc2`: Es el nombre de usuario de Windows en el PC 2 (puedes verlo ejecutando `whoami` en el PC 2).
  * `IP_del_pc2`: La IP que anotaste en el paso anterior (ej. `192.168.1.15`).
    
**2. La primera vez te preguntará si confías en el equipo. Escribe `yes` y presiona Enter.**

**3. Introduce la contraseña del PC 2.**

¡Listo! Tu terminal ahora estará controlando el PC 2.
------------------------------
## Paso 3: Acceder a los directorios del PC 2
Una vez conectado por SSH, ya estás dentro del PC 2. Para moverte por sus carpetas y ver los archivos usa estos comandos básicos:

* Ver dónde estás parado: Escribe `pwd`. Por defecto entrarás en tu carpeta de usuario (`C:\Users\TuUsuario`).
* Listar archivos y carpetas: Escribe `dir` (en PowerShell) o `ls` (si el PC 2 usa SSH de Linux).
* Moverte a otra carpeta (ej. el Escritorio):
```c
cd Desktop
```

* Ir a la raíz del disco C:
```c
cd C:\
```
------------------------------
## ¿Quieres ver los archivos de forma visual? (Bonus)
Si en lugar de usar la terminal en el PC 1 quieres ver las carpetas del PC 2 en el Explorador de Archivos visual, no necesitas SSH.
En el PC 1, abre el Explorador de archivos, haz clic en la barra de direcciones arriba, escribe \\IP_del_pc2\c$ (ejemplo: \\192.168.1.15\c$) y presiona Enter. Te pedirá el usuario y contraseña del PC 2 y verás todo el disco C de forma gráfica.
Para ayudarte a dejar la conexión perfecta, dime:

* ¿El PC 2 es un Windows normal o quieres conectar directamente dentro de la máquina virtual de WSL del PC 2?
* ¿Te ha aparecido algún error de conexión o de firewall al intentar el comando?
* ¿Necesitas copiar archivos de un PC al otro usando la terminal?
